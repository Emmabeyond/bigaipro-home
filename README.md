## 快速开始

1. 注册并登录 BigAiPro 账户
2. 获取您的 API Key
3. 通过我们的标准 API 接口调用 AI 服务

```bash
# 示例请求
curl -X POST "https://api.bigaipro.com/v1/chat/completions" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-4",
    "messages": [{"role": "user", "content": "你好，请介绍一下你自己"}],
    "temperature": 0.7
  }'
```
